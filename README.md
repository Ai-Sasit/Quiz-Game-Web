# Web Development | Mini Project

  - Project name: Quiz Game Web
## Tools
  - Docker
  - ReactJS
## Docker Container
  - MongoDB
  - MongoDB Express
## Dependencies
  - Babel
  - Nodemon
  - Express
  - MongoDB
  - Axios
## How to run!
  - Both frontend and backend
  ```
  npm i
  ```
  ```
  npm start
  ```
  - Docker start
  ```
  docker-compose up --build
  ```
  - Docker stop
  ```
  docker-compose down
  ```
